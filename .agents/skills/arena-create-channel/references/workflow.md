# New Channel Workflow

## Default sequence

1. Restate the theme in a sharper form if the user prompt is vague.
2. Pick a channel title and short markdown description.
3. Search the web for diverse source types.
4. Search Premium Are.na for adjacent blocks and channels worth connecting.
5. Build a 12-item manifest with a direct core and a few tangent edges.
6. Execute through `$arena-api-operator`.

## Defaults

- Visibility: `closed`
- Execution: immediate write
- Preview: only when the user explicitly asks for a preview or dry run

## Good additions

- A small original text block that frames the theme
- One or more images
- One or more links to articles or webpages
- At least one attachment candidate such as a PDF when available
- At least one embed candidate such as a YouTube or Vimeo URL
- One existing Are.na block or channel when Premium search surfaces a strong fit

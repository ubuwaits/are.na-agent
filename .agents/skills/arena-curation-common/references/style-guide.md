# Are.na-native Curation Style

Treat the channel like a public notebook, not a polished article.

## Mix

- Keep most items directly on-theme.
- Add a smaller number of tangential items that widen the mood or adjacent ideas.
- Add at least one playful or low-brow item when it fits: meme, odd forum post, throwaway image, ephemera.
- Balance high and low sources: quote, PDF, lecture, article, image, video, oddity.

## Defaults

- New channels: 12 items unless the user asks otherwise.
- Existing channels: 6 additions unless the user asks otherwise.
- Default new-channel visibility: `closed`.

## Block type goals

Aim to include all supported block types when feasible from the available material:

- `Text`
- `Image`
- `Link`
- `Attachment`
- `Embed`

Do not force the mix if the theme or available sources make one type unnatural.

## Dedupe

- Avoid repeating the same URL under tracking variants.
- Avoid reconnecting existing Are.na blocks/channels already present in the target.
- Avoid adding two items that make the same point in the same tone unless repetition is clearly intentional.

## Voice

- Prefer concise, observant text blocks over long explanations.
- Let juxtaposition do part of the work.
- Preserve a slightly wandering notebook feel rather than over-curating into a rigid syllabus.
